## ✨ Key Features

* **Auspicious Items Search:** Search and discover auspicious items specifically tailored to your birth date.
* **Store Locator (Map Integration):** Find nearby locations or stores to rent and purchase specific auspicious items using the integrated map.
* **Categorized Wallpapers:** Explore and download auspicious wallpapers categorized by birth dates, zodiac signs, and specific luck enhancement themes.
* **Random Wallpaper Generator:** Get a randomly selected auspicious wallpaper to boost your luck.

## 🛠️ Tech Stack & API

* **Framework:** Flutter (Android/iOS)
* **Programming Language:** Dart
* **API Integration:** Google Maps API (for store location mapping)

## 🚀 How to Run (Optional)

1. Clone the repository: `git clone https://github.com/WorawarunPl/KhongMongKol-Application.git`
2. Change directory: `cd KhongMongKol-Application`
3. Install dependencies: `flutter pub get`
4. Run the app: `flutter run`
